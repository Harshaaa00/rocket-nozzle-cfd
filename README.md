## 📋 Overview

This project presents the complete design and CFD simulation of a **convergent–divergent (C-D) rocket nozzle** producing approximately **600 N of thrust**. The nozzle geometry was derived from compressible flow theory and numerically validated using ANSYS Fluent with a k–ω SST turbulence model.

---

## ⚙️ Design Conditions

| 🔩 Parameter            | 📋 Value         |
|------------------------|-----------------|
| **Chamber Pressure**   | 1.8 MPa         |
| **Chamber Temperature**| 3000 K          |
| **Gas Model**          | Ideal Gas       |
| **Turbulence Model**   | k–ω SST         |

---

## 📐 Nozzle Geometry

| 📏 Dimension           | 📋 Value         |
|-----------------------|-----------------|
| **Chamber Diameter**  | 48 mm           |
| **Throat Diameter**   | 16 mm           |
| **Exit Diameter**     | 36.6 mm         |
| **Total Length**      | 226 mm          |
| **Area Ratio**        | 5.23            |

---

## 📊 CFD Results

| 🔬 Parameter                  | 📈 Result                      |
|------------------------------|-------------------------------|
| **Flow Condition at Throat** | ✅ Choked flow achieved        |
| **Exit Mach Number**         | ≈ **2.8**                     |
| **Pressure Drop**            | 1.2 MPa → 0.1 MPa             |
| **Temperature Drop**         | 3300 K → 1700 K               |

### 🔍 Key Observations

- 🔒 **Choked flow** successfully achieved at the throat
- 💨 **Supersonic exit** conditions reached with Mach ≈ 2.8
- 📉 **Pressure** drops from 1.2 MPa to 0.1 MPa across the nozzle
- 🌡️ **Temperature** drops from 3300 K to 1700 K confirming energy conversion to kinetic energy
